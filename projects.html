Guess!

A simple high-low guessing game. Initial project causing interaction between javascript, jQuery and the DOM. In addition to a simple message letting the player know if they're higher or lower than the target number, the background color and brightness changes based on the distance of the guessed number from the target number.

  • Client: N/A, personal project
  • Date: February 2016
  • Service: HTML, CSS, Javascript, jQuery
  • GitHub: https://github.com/nbidler/guess.git
  • Live Demo

Tic-tac-toe

First group project, a simple Tic-Tac-Toe game. Options include a 3x3 or 5x5 grid, a choice of different player markers (that prevents players from choosing identical markers), sound effects, and a game mode option that lets players choose an unused square normally or attempt to 'steal' the other player's squares with a 65% chance of success (otherwise, the turn is wasted).

  • Client: N/A, group project
  • Date: February 2016
  • Service: HTML, CSS, Bootstrap, Javascript, jQuery
  • GitHub: https://github.com/nbidler/hacka-tictactoe-athon.git
  • Live Demo

Search by location

A hackathon group project, with the use of APIs as the objective. Accepts a search term and a click on the map, gives back pictures from Flickr and tweets from twitter. Of note: Flickr has geotagging photos on by default and Twitter has geotagging off by default - the search system only returns results with a valid location.

  • Client: N/A, group project
  • Date: March 2016
  • Service: Made with HTML, CSS, Bootstrap, Javascript, AJAX, jQuery, and Google Maps/Flickr/Twitter APIs.
  • GitHub: https://github.com/Mayrod1/C7_Hackathon.git
  • Live Demo

StoryPress

A hackathon group project, attempting integration of two teams: one for front-end and one for back-end. Users create an account, with which they can make creative writing prompts or create stories that show their prompts. Guest (account-less) users can view prompts and stories but cannot interact with them.

  • Client: N/A, group project
  • Date: March 2016
  • Service: Made with HTML, CSS, Bootstrap, Javascript, AJAX, jQuery, and PHP.
